Win7sé v0.3
-----------
Author: vhanla (Victor Alberto Gil)
http://apps.codigobit.info

Win7sé is an exposé wannabe utility for Windows 7 that shows the Alt-Tab 
Switcher using mouse gestures.

Features:
- Shows the AltTab Switcher by moving the mouse cursor to any of the following 
  hot spots  Top Left, Top Right, Bottom Left and/or Bottom Right user choice
- Run at Windows logon
- Switch between the last applications with one gesture
